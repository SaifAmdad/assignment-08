## Tiles Gallery

Showing Tiles and its details. Loged in user can visite every route and update Name and Photo-url.

Live link : https://assignment-08-olive.vercel.app/

### Some Features

- User can register with credentials.
- Signin with google.
- Login with Email & Password

## Tech Stack

**npm:** better-auth, react-fast-marquee, react-toastify, swiper

- For using - First, clone git-repo, set env and follow:

```bash
npm install
&
npm run dev

```
